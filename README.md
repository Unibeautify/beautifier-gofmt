> Gofmt beautifier for Unibeautify

## Installation

```bash
npm install --global @unibeautify/beautifier-gofmt
```

Then change `.unibeautifyrc.yml`:

```yaml
Go:
    beautifiers: ["Gofmt"]
```

See [`unibeautify-cli`](https://github.com/Unibeautify/unibeautify-cli) for details.
